# printer2D

A python script to intercept and work with clawPDF print jobs.

## Note

The `PDFtoPrinter.exe` file is downloaded from [here](https://mendelson.org/pdftoprinter.html) and is placed inside this directory. Its size is around 11 MB.

Also, [clawPDF](https://github.com/clawsoftware/clawPDF) has to be installed.

Python version 3.12.3 is preferred to avoid PIP package installation problems.
